# 🔵 CSS Task 2 — Form Styling

### 🎯 Goal
Style the user profile form.

### 🧠 Class naming

In this task, use `.class` to style elements.

Use simple component-based naming:
`component-name + element`

Examples:
- `profile-form`
- `profile-form-input`

---

### 📦 Requirements

#### 🧩 Form container
- Style the form as a card
- Add background color `#fff`
- Add padding `16px`
- Set width `250px`
- Set min height `320px`
- Add border radius `10px`

---

#### 🧱 Form structure
- Arrange fields in a vertical column
- Add spacing between fields `16px`

---

#### ✏️ Inputs
- Add padding inside inputs
- Add light border `#ccc`
- Add border radius `3px`

---

#### 🔘 Buttons
- Remove default border from buttons
- Add padding to buttons
- Add border radius `3px`
- Buttons should have a pointer cursor

#### 🎨 Button styles
- Submit button:
  - background color `#C4D9FF`
  - hover background `#AECBFC`
  - active background `#95BDFC`

- Reset button:
  - background color `#FFC4C4`
  - hover background `#FCB6B6`
  - active background `#FFA6A6`

---

### 💡 Hints
- Use spacing (`margin` or `gap`) to separate elements
- Style interactive states using `:hover` and `:active`
- You may need:
  - `padding`
  - `margin`
  - `border`
  - `border-radius`
  - `background-color`

---

### 👀 Check yourself
- Does the form look like a card?
- Do buttons look interactive?
- Do hover effects work?
