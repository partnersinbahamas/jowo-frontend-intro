# 🔵 CSS Task 3 — Profile Card

### 🎯 Goal
Style the user profile card.

---

### 📦 Requirements

#### 🧩 Card
- Style the card as a UI component
- Add background color `#fff`
- Add padding
- Add border radius

---

#### 👤 Header
- Image, name, and email should be visually grouped
- Content should be centered

---

#### 📝 Content
- Style user name with color `#411B4F`
- Style skills and description text with color `#612978`
- Use different font sizes

---

#### 🔘 Link
- Style the message link as a button
- Place the button at the bottom of the card
- Add hover effect

#### 🎨 Link styles
- Default
  - background color `#fff`
  - color `#612978`
  - border color `#612978`

- On hover
  - background color `#612978`
  - color `#fff`
